## 2. 阶段一：修复 TokenDumper（1-2天）

### 2.1 修复三引号字符串解析

**问题**: `MatchStringToken` 方法在处理转义字符时逻辑不正确。当前代码检查 `currentLine[end - 1] == '\\'` 但没有正确处理连续转义的情况。

**修复方案**:

```csharp
// 当前代码 (TokenDumper.cs:223-228)
var end = currentLine.IndexOf(quotes, start, StringComparison.Ordinal);
if (end > 0 && currentLine[end - 1] == '\\')
{
    content.Append(currentLine.AsSpan(start, end + 1 - start));
    start = end + 1;
    continue;
}

// 修复后：计算转义字符数量
var end = currentLine.IndexOf(quotes, start, StringComparison.Ordinal);
if (end > 0)
{
    // 计算引号前的反斜杠数量
    int backslashCount = 0;
    for (int i = end - 1; i >= 0 && currentLine[i] == '\\'; i--)
        backslashCount++;
    
    // 如果反斜杠数量为奇数，说明引号被转义
    if (backslashCount % 2 == 1)
    {
        content.Append(currentLine.AsSpan(start, end + 1 - start));
        start = end + 1;
        continue;
    }
}
```

**修改文件**: `src/PyRebuilderSharp.Core/Testing/TokenDumper.cs:220-230`

### 2.2 修复特殊字符和 Unicode 解析

**问题**: 当前 `WordRegex` 只匹配 ASCII 字母数字，无法处理 Unicode 字符如 ∞（无穷符号）。

**修复方案**:

```csharp
// 当前代码 (TokenDumper.cs:19)
[GeneratedRegex(@"[A-Za-z_][A-Za-z0-9_]*")]
private static partial Regex WordRegex();

// 修复后：支持 Unicode 标识符
[GeneratedRegex(@"[\p{L}_][\p{L}\p{N}_]*")]
private static partial Regex WordRegex();
```

**修改文件**: `src/PyRebuilderSharp.Core/Testing/TokenDumper.cs:19`
